# O que é uma classe?
- Nome da classe 
- Conjunto de atributos (descrição)
- Conjunto de métodos (comportamento)

# Exemplo
- Classe: Cachorro 
- Conjunto de atributos: Altura, Peso, Temperamento, Inteligencia, Tempo de vida... 
- Conjunto de métodos: Latir, Dormir, Treinar... Verbos ---> Ações, são comportamentos de uma classe

# Polimorfismo
- Habilidade de variáveis terem "mais de um tipo"

# O que são variáveis
- Área de memória associada a um nome, que pode armazenas valores de um determinado tipo.

# Como declarar uma variável em java
- <tipo> <nome da viariável>;
- <tipo> <nome da viariável> =  <valor>;

# Convenção de nomenclatura
- Sempre verificar as palavras reservadas - pois não podem ser utilizadas em nome de variável.
- a-z A-z _ & (primeira letra)
- a-z A-z _ & 0-9 (da segunda letra em diante)
- camelCase - o nome da variável é recomendado começar com minúscula.
- sempre dar nome a variáveis ao que realmente está se referindo

# Controle de decisão
# Comando IF
- Comandos dentro de um bloco IF somente são executadas se a condição for verdadeira.
- Se a condição for falsa, os comandas não são executados.

# Comando IF - ELSE
- Comandos dentro de um bloco ELSE somente são executados se a condição IF for falsa.

# Múltiplos IF - ELSE
- É possível ter vários if-else aninhados! 

# Switch - Case
- Funciona como multiplos If-Else (Alternativa de if-else aninhados)
- SWITCH (Seria o if);
- CASE é como se fosse comparando com o que tem dentro da expressão;
- BREAK encerra o switch.
