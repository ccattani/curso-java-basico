# Vantagens de Orientação a Objetos
- Reuso do Código
- Reflete o mundo real
- Facilita a Manutenção no Código

# Conceitos
- Classes
- Objetos
- Herança
- Polimorfismo

# Classe
- Descrição de um grupo de objetos
- Ex: Você pode ser definido como um objeto no mundo da programação
- Ex: Cachorro - pode ser definido como um objeto? Sim, pode

# O que é uma classe?
- Nome da classe 
- Conjunto de atributos (descrição)
- Conjunto de métodos (comportamento)

# Exemplo
- Classe: Cachorro 
- Conjunto de atributos: Altura, Peso, Temperamento, Inteligencia, Tempo de vida... 
- Conjunto de métodos: Latir, Dormir, Treinar... Verbos ---> Ações, são comportamentos de uma classe

# Herença
- Permite reutilização da estrutura e do comportamento de uma classe.

# Polimorfismo
- Habilidade de variáveis terem "mais de um tipo"
